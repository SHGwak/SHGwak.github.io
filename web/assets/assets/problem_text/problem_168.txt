지면에서 시간 0 일 때 질량 $2 \mathrm{~kg}$ 인 물체를 연직 위방향으로 $20 \mathrm{~m} / \mathrm{s}$ 의 속력 으로 발사하였다. 물체가 지면에 도달할 때까지 (시간 $t$ - 퍼텐셜에너지 $U$ ) 사이의 관계를 그래프로 나타내어라. 단, 물체가 지면에 있을 때 퍼텐셜에너지는 0 이다.
