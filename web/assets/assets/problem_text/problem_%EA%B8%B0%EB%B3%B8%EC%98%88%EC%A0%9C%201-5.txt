시간 $t=0$ 일 때 정지하고 있던 어떤 물체가 직선운동을 시작하여 $t=5 \mathrm{~s}$ 일 때 동쪽으로 $10 \mathrm{~m} / \mathrm{s}$ 의 속력으로, $t=10 \mathrm{~s}$ 일 때 서쪽으로 $10 \mathrm{~m} / \mathrm{s}$ 의 속력으로 운동하 였다.

(1) $t=0$ 부터 $t=5 \mathrm{~s}$ 까지 물체의 평균가속도의 크기와 방향을 구하여라.

(2) $t=5 \mathrm{~s}$ 부터 $t=10 \mathrm{~s}$ 까지 물체의 평균가속도의 크기와 방향을 구하여라.
